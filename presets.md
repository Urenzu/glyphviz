# Presets

Press `p` while running the viz to cycle presets defined in `src/presets.ts`. Set `startPresetName` in `src/main.ts` to any preset name (or `"custom"`) to boot directly into it.

Current presets:
- `industrial-amber`: military/industrial orange with white highs.
- `cold-radar`: icy blue radar grid.
- `stealth-green`: minimal green terminal.
- `thermal`: heat-map inspired amber/red.
